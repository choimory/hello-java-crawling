# 개요

- Java로 크롤링을 해보자

# 사용할 크롤링 스택

## BeautifulJsoup

- 파이썬의 BeautifulSoup의 Java 버전이다.
- 텍스트를 그대로 긁으므로 빠르다.
- 대신 동적 렌더링 결과에는 대응할 수 없다.

## Java Selenium

- 크롬드라이버를 이용해 자바스크립트 동적 렌더링 된 결과에도 대응할 수 있다.
- 대신 느리다.

# 크롤링 목표

- 하루 주기 스케줄링
- 특정 쇼핑 사이트의 특정 상품 주소 내의 가격태그를 크롤링한다
- 가격 확인하여 목표치 이하일 경우 Slack 발송
